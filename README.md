# SocialMediaApp
# To run the code 

go to server
```
cd server
```
Install all the necessary dependencies
```
npm install or npm i
```
create a MONGODB project and add it to the URI

start the server
```
npm start

```
# After starting the server

go to client
```
cd server
```
Install all the necessary dependencies
```
npm install or npm i
```
start the client
```
npm start

```
